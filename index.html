I was in charge of designing and implementing the UI for the game, it was a lot of late nights but a tons of fun. I also helped get the games multiplayer set up by connecting clients to a server that our team hosted off-campus.

Procedural Content Generation

This was purely a passion project. I loved playing Minecraft as a kid (I still play here and there) and I always wondered how the Minecraft world would continuously expand and generate new land. To answer my own question I started doing some research and came across the fascinating world of procedural generation. I wanted to put my new found knowledge to work so I built this. It is a small project I build in Unity that generates pseudorandom landmass with Perlin Noise and mesh generation.

Check it out!

Algorithm Visualization

This was a project I built to help me get a better understanding of some common shortest path algorithms. I figured the best way to understand would be to visualize it. The project was super fun! I used Python and PyGame to build a tool that finds and visualizes the shortest path between two points on a grid that you choose. I can find the shortest path the using A*, BFS, or Dijkstra's algorithms.

Check it out!
